A simple implementation of an autonomous agent which exploits OpenCV to play the famous dino game

## Installation

You can install it from pypi

```bash
pip install dino-skynet
```

## Usage

### Try the game

To try the game

```bash
python -m dinoskynet
```

press `spacebar` to start the game
